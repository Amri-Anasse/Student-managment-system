# 🎓 Student Management System

## 📌 Description
This project is a web-based student management system built using PHP and MySQL.  
It allows you to add, edit, delete, and view students.

## ⚙️ Technologies Used
- PHP (MVC structure)
- MySQL
- HTML / CSS(bootstrap)

## 🚀 Features
- Add new student
- Edit student information
- Delete student
- List all students

## 🛠️ Installation
1. Clone the repository
2. Import the database (ecole.sql)
3. Run the project using XAMPP or WAMP
